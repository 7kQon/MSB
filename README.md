## Environment

### Unity

Version 2019.2.1f

Essential Packages
- Cinemachine
- PostProcessing

Essential Assets
- CorgiEngine
- UnityMainThreadDispatcher
- SpriteOutline

### ProudNet with WindowsOS

ProudNet : Version 1.7.47302  
WindowsOS : DotNetFramework 4.6  

### Android SDK

Targets SDK 29 (Android 10)  
Supports SDK 21 (Android 5)
